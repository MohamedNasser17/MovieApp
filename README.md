### Architecture
 - **MVVM** is used for every module architecture
 - **Monolith** is used for project architecture.


### Network
 - **URLSession** is used to handle network calls.
 - **AsyncImage** is used to fetch the images in async way.

### UI Framework
 - **SwiftUI** is used to create all views.
